# **App Name**: LedgerSync

## Core Features:

- Income Tracking: Record daily income entries, categorizing them for better analysis.
- Monthly Income Charts: Display monthly income trends using interactive charts.
- Account Management: Manage individual accounts, adding and subtracting amounts as money is received.
- Admin Dashboard: Provide an admin interface to view aggregated and individual user income.
- Client Account Creation: Allow admin to create and manage client person accounts.
- Product Management: Enable the admin to add products with prices that can vary per client account.
- Bill Generation & Sharing: Create detailed bills with product and quantity, allowing price adjustments at bill time and options for sharing the bill as an image, inclusive of previous balance and sum of current bill amount.
- Data Persistence: All data is saved and retrievable from Firestore and it may see on different devices.
- Bill Sharing Content Suggestion Tool: AI tool suggests text content for bill image sharing to the user to customize further.
- Exporting tool: Enable exporting income charts and data to PDF or Excel formats.

## Style Guidelines:

- Primary color: Deep teal (#468499), evoking trust and reliability in financial management.
- Background color: Light teal (#E0F8FF), offering a clean and calming backdrop.
- Accent color: Muted gold (#B8860B), providing a touch of sophistication for important actions.
- Body and headline font: 'PT Sans', sans-serif font for clear and accessible text.
- Use financial icons that clearly represent income, expenses, and reports.
- Design a dashboard with a clear, intuitive layout that provides quick access to key metrics and features. Use a tabbed interface for easy navigation between sections like 'Income,' 'Expenses,' and 'Reports.'
- Subtle animations to transition between data views, creating a dynamic yet non-distracting user experience.
